## 자바를 이용해서 컴퓨터 프로그램 만들기
### 자바 코드 작성시 기본 규칙
1. 자바에서는 ;을 기준으로 문장을 구분한다.
2. 컴파일러가 알 수 없는 기호들은 변수로 선언해 준다. 변수는 **데이터 타입과 변수명;** 형식으로 한다.
